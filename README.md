# marjunmonteronbriones

## 🌐 Portfolio App

## 📖 Description

This is a personal portfolio web application built using Streamlit. It is designed to showcase my profile, skills, projects, and contact information in a simple, interactive, and visually appealing way.

The app serves as a digital resume where visitors can explore my background and the work I’ve done.

---

## ✨ Features

### 🏠 Home

Provides a welcoming introduction and overview of the portfolio.

### 👤 About

Displays personal background, information, and profile details.

### 🛠 Skills

Highlights technical skills, tools, and areas of expertise.

### 📂 Projects

Showcases completed projects and works with descriptions.

### 📞 Contact

Offers ways to get in touch, including social links or email.

### 🖼 Image Integration

Enhances the interface with personalized images and visuals.
